## input type

```html
<input type="text" placeholder="type here">
```
text = 텍스트 상자
placeholder = 상자 바탕에 쓰여짐

---

### password
---

```html
<input type="password">
```
텍스트 상자의 내용이 가려짐

---

### checkbox
---

```html
<input type="checkbox">
```
체크박스

---

### radio
---

```html
<input type="radio" name="choice" value="a"> a
<input type="radio" name="choice" value="b"> b
```
체크박스와 같으나 radio는 둘 중 하나만 선택 가능
name이라는 속성을 이용해서 하나로 묶어주었다.

---

### textarea
---

```html
<input type="textarea">
```
줄바꿈이 가능한 텍스트 상자

---
