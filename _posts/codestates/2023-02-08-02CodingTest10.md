### 구하는 방법(가장 작은 수들의 곱)

```
12 = 2 x 2 x 3
18 = 2 x 3 x 3

2 x 3 x 3 x 2 = 36
```

겹치는 부분인 2와 3을 곱한 수인 6이 최대공약수

6을 중심으로 2와 3을 곱해 나오는 수인 36이 최소공배수
