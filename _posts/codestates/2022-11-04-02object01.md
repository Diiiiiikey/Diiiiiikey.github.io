## dot notation

```js
let user = {
    firstName: "Steve",
    lastName: "Lee",
}

user.firstName; // 'Steve'
```

## braket notation

```js
user['lastName']; // 'Steve'
```

## 추가

```js
user['city'] = 'Seoul';

user.isAdult = true;
```

## 삭제

```js
delete user.isAdult;
```

## 확인

```js
'isAdult' in user; // false
```
