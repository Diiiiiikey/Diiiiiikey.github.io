## function

```js
function sayHello(name){
    console.log(Hello, ${name});
}

sayHello("DK");
```
function은 함수 <br/>
sayHello 함수명 <br/>
name 매개변수(parameter) <br/>
console.log(Hello, ${name}) 함수의 실행코드 <br/>
sayHello("DK"); 호출 방법 이때 ()안을 인자(argument) <br/>
<br/>
<br/>
