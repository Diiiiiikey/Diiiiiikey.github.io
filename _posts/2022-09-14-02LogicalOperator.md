## 논리연산자

<html>
|| <br/>
or: 하나라도 true 면 true, 첫번째 true를 발견하면 즉시 평가를 멈춤 <br/>

                a || b 

<br/>
<br/>

&& <br/>
And: 하나라도 false 면 false, 첫번째 false를 발견하면 즉시 평가를 멈춤 <br/>

                a && B 

<br/>

! <br/>
Not: true 와 false의 반대값을 출력 <br/>

                !a 

<br/>
</html>
